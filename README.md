# Фриланс-биржа (WPF-приложение)

Desktop-приложение для взаимодействия заказчиков и фрилансеров. Позволяет публиковать заказы, просматривать каталог, регистрироваться и сохранять данные между сеансами.

## Стек технологий

- **C#** — бизнес-логика
- **WPF (Windows Presentation Foundation)** — платформа для desktop-приложения
- **XAML** — разметка и стилизация интерфейса
- **JSON + Newtonsoft.Json** — хранение данных
- **Frame + Page** — навигация между страницами

## Функциональные возможности

| Роль | Возможности |
|------|-------------|
| **Гость** | Просмотр каталога заказов, просмотр деталей заказа |
| **Заказчик** | + создание новых заказов (название, описание, изображение, цена, срок) |
| **Фрилансер** | + просмотр всех заказов |
| **Любой пользователь** | Регистрация / вход в аккаунт |

### Ключевые функции

- Каталог заказов в виде карточек (изображение, название, цена, автор, рейтинг, срок)
- Регистрация с валидацией: проверка пустых полей, корректность email, подтверждение пароля
- Создание заказа с выбором изображения с компьютера (предпросмотр)
- Детальный просмотр заказа при клике на карточку
- Сохранение данных в JSON (пользователи и заказы не пропадают после закрытия)
- Единая стилизация: эффекты при наведении/нажатии на кнопки

## Архитектура приложения
MainWindow (окно с Frame и шапкой/подвалом)
├── MainPage (главная страница с каталогом)
├── Registracia (страница регистрации)
├── LoginPage (страница входа)
├── WorkerPage (создание заказа)
└── OrderDetailsPage (детальный просмотр)

text

- **Frame** — контейнер для навигации между страницами
- **Шапка и подвал** вынесены в MainWindow и видны на всех страницах
- Данные загружаются из `data.json` при запуске, сохраняются при закрытии
